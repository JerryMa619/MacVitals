Privacy Policy

MacVitals keeps monitoring local.

Last updated: June 29, 2026

MacVitals is designed as a local macOS system monitor. The current app MVP does not include accounts, in-app analytics, advertising SDKs, tracking SDKs, or diagnostic upload.

Data MacVitals Reads

MacVitals reads local system status such as memory pressure, CPU load, disk free space, battery status, network throughput, thermal state, and visible running app memory usage. This information is used to display live vitals and local recommendations.

Data Collection

MacVitals does not collect personal data on a server. Monitoring data stays on your Mac. Diagnostic snapshots are copied to your clipboard only when you explicitly choose that action.

Website Analytics

The MacVitals website uses Plausible Analytics to understand aggregate website traffic and button clicks, such as GitHub, Early Access, Privacy, and Support clicks. Plausible does not use cookies and is used only for website analytics, not in-app monitoring data.

Network Use

The current MVP does not require network access for monitoring. MacVitals does not upload system metrics or diagnostic snapshots.

User Control

MacVitals does not automatically force-quit apps, delete files, or clear caches. It may open Activity Monitor so you can decide what to do.

Contact

For privacy questions, open an issue at github.com/JerryMa619/MacVitals/issues.
