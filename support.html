Common Questions

Does MacVitals clean memory automatically?

No. MacVitals focuses on monitoring and diagnosis. It shows pressure, explains likely causes, and can open Activity Monitor for user-controlled action.

Does MacVitals upload my system data?

No. The MVP does not include analytics, account sync, tracking, or diagnostic upload.

Why does thermal status not show a temperature in Celsius?

MacVitals uses Apple's public thermal state API for App Store safety. It avoids private sensor APIs that could create review or compatibility issues.

Troubleshooting

  • Use the Refresh button if the dashboard has just opened.
  • Open Settings to switch sampling mode between Balanced, Low Power, and Live.
  • Use Copy Diagnostic Snapshot when reporting a performance issue.
  • Open Activity Monitor if MacVitals highlights a high-memory app.
